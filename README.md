# StoicizmFrame v3.14_clean

## Описание проекта
StoicizmFrame — это архитектурный узел, посвящённый созданию уникального философского и прикладного контента.
Каждый элемент проекта оформляется как артефакт: документ, видео, сценарий или кодовый модуль.
Цель — сохранить и передать стоическую традицию и прикладную мудрость будущим поколениям.

## Структура узла
- **README.md** — общий паспорт проекта
- **TELEGRAM.md** — описание канала Telegram
- **YOUTUBE.md** — описание канала YouTube
- **scenarios/** — сценарии видео и публикаций
- **src/** — кодовые модули (`donor_parser.py`, `scenario_builder.py`, `layout_composer.py`, `voice_adapter.py`)
- **tools/** — служебные скрипты для архитектурной гигиены

## Архитектурные узлы
- **ENTRY** — входные сцены и посты с кинематографическими заставками
- **LEGACY** — документальные материалы, фиксирующие философские и культурные артефакты
- **CLIENT** — адаптированные подборки для широкой аудитории

## Публикационные каналы
- **Telegram:** подборки текстов, цитаты, практические советы, короткие видео
- **YouTube:** нарративные видео до 7 минут, компиляции 14–21 мин., тематические плейлисты

## Техническая интеграция
- Озвучка: уникальный голос стоического старца (Bark / Coqui XTTS + ffmpeg постобработка)
- Визуализация: Stable Diffusion, AnimateDiff, Deforum
- Маршрутизация: `layout_composer.py`, `voice_adapter.py`
- Кодировка: UTF‑8 LF, без BOM
- Контроль гигиены: `run_hygiene.ps1`, `.gitattributes`, `normalize_yaml.ps1`

## Цели
- Создание уникального философского и прикладного контента
- Документальное оформление каждого узла как артефакта
- Систематизация публикаций в Telegram и YouTube
- Сохранение и передача традиции StoicizmFrame будущим поколениям
