# Кодировка: UTF-8-LF-без BOM

## Правила
1. Все файлы сохраняются в кодировке UTF-8 LF без BOM.
2. Черновики → drafts/, финализация → finalized/.
3. Компиляции фиксируются в docs/LIST_OF_ORIGINS.md.
4. Перед любыми изменениями обновляется ROLLBACK.md.
5. Узел не архивируется до порога v10.0.
6. Закрытие узла не является архивированием. 
   Закрытие фиксирует завершение логического цикла работ (создание сцен, отчётов, ROLLBACK и т.д.), 
   но узел остаётся изменяемым до достижения порога v10.0. 
   Тег служит контрольной точкой для восстановления, но не блокирует внесение правок. 
   Любые изменения допускаются при условии обновления ROLLBACK.md и соблюдения кодировки UTF‑8 LF без BOM.
