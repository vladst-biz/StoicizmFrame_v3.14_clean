# LEGACY_GEN_001 — Наследие тестового блока

## Архитектурные решения, ставшие постоянными
- Строгая структура узла: drafts/, finalized/, compiled/, docs/
- Обязательное использование UTF-8 LF без BOM для всех файлов
- Автоматическая сборка компиляций через единый сценарий
- Обязательное наличие PROTOCOL.md, README.md, ROLLBACK.md, DO_NOT_ARCHIVE
- Проверка кодировки перед фиксацией
- Ручная валидация компиляций как часть тестового цикла

## Практики, вошедшие в PROTOCOL
- Создание отчётов через StreamWriter (LF, без BOM)
- Обязательная проверка читаемости файлов в VS Code
- Обязательная фиксация маркеров отката перед пересборкой
- Строгая последовательность: сцены → компиляции → отчёты → ROLLBACK

## Значение блока
GEN_001 подтвердил работоспособность фабрики StoicizmFrame и стал фундаментом для всех последующих узлов.
