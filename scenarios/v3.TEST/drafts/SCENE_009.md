- - StoicizmFrame-v3.TEST--Draft
- - Кодировка: UTF-8-LF-без BOM

## Сцена 009 — спокойствие среди хаоса
- Доноры: Марк Аврелий «Размышления», Эпиктет «Энхиридион», наблюдения за современными стрессовыми ситуациями.
- Сбор: scenario_builder.py (выделение примеров хаоса, эмоциональных реакций, стоических ответов).
- Компоновка: layout_composer.py (визуальные метафоры хаоса, плавные переходы, минимализм).
- Озвучка: voice_adapter.py (спокойный, ровный голос, акцент на внутреннем равновесии).
- Заявка: Foundry+ видео 7 минут, стиль «спокойствие», без лиц, с визуальными символами порядка.

## Нарратив
- «Хаос вокруг нас не обязан становиться хаосом внутри нас.»

## Структура сцены
1. Хаос — внешняя ситуация, создающая давление.
2. Реакция — естественная эмоция человека.
3. Принцип — стоическое правило, которое возвращает к центру.
4. Применение — как этот принцип работает в моменте.
5. Завершение — спокойная, уверенная мысль.

## Технические параметры
- Формат: Markdown, UTF-8-LF-без BOM.
- Длина: 120–180 слов.
- Стиль: спокойный, ровный, минималистичный.
- Проверка: корректное отображение в VS Code и PowerShell 7.
