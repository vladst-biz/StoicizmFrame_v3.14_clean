# Кодировка: UTF-8 LF без BOM

## Узел v3.15 — Evolution
Узел v3.15 представляет собой эволюционный цикл фабрики StoicizmFrame. 
Он включает развитие нарратива, визуала, озвучки и интеграции мультимодальных модулей.

## Состав узла
- SCENE_004 — базовый паспорт сцены.
- SCENE_005 — эволюция нарратива.
- SCENE_006 — эволюция визуала.
- SCENE_007 — эволюция озвучки.
- SCENE_008 — интеграция всех компонентов.

## Архитектурные принципы
1. Все файлы — UTF-8 LF без BOM.
2. Все изменения фиксируются в ROLLBACK.md.
3. Узел не архивируется до порога v10.0.
4. Сцены маршрутизируются через layout_composer.py.
5. Озвучка проходит через voice_adapter.py.
6. Визуал формируется через Stable Diffusion / AnimateDiff / Deforum.
7. Финальная сборка — заявка в Foundry.
