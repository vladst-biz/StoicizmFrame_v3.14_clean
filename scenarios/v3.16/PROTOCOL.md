# PROTOCOL — v3.16 Institution

## Назначение
Локальный протокол узла v3.16 фиксирует правила работы,
структуру файлов, порядок документирования и принципы наследования.

## Структура узла
- SCENE_001.md
- SCENE_002.md
- SCENE_003.md
- SCENE_004.md
- SUMMARY.md
- LEGACY.md
- EPILOGUE.md
- PROTOCOL.md
- ROLLBACK.md
- DO_NOT_ARCHIVE

## Правила узла
1. Все файлы узла находятся строго в папке v3.16.
2. Корень проекта остаётся чистым.
3. Документы создаются в UTF‑8 LF без BOM.
4. Узел не архивируется до v10.0.
5. Любые изменения фиксируются через git с пометкой узла.
