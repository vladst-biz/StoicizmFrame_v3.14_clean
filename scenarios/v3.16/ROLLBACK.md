# ROLLBACK — v3.16 Institution

## Назначение
Маркер отката для узла v3.16.
Позволяет безопасно вернуться к состоянию узла
после завершения всех файлов и сцен.

## Точка отката
- После создания всех 7 архитектурных файлов.
- После проверки кодировки и структуры.
- После фиксации git-коммитом.

## Инструкция
В случае ошибки:
1. Выполнить git reset --hard <commit_id>.
2. Проверить структуру узла.
3. Повторить генерацию файлов по протоколу.
