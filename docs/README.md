# StoicizmFrame v3.0

StoicizmFrame — архитектурный проект по генерации уникального философского контента.
Цель: создание и публикация видео и текстов о стоицизме через облачные ресурсы (Azure Foundry, OpenAI).
Структура: генератор → рефреймер → SocialFrame → архив.
Каналы: YouTube, Telegram.
Документация: README.md, MANIFEST.md, PROTOCOL.md, LEGACY.md.
